# Playwright Test Automation Framework

This repository contains a UI and API automation suite built with **C#** and **Playwright**.

---
## Prerequisites

Before running the tests, ensure you have the following installed:

1. **.NET SDK:** [Download .NET 8.0](https://dotnet.microsoft.com/download)

---

## Installation

	git clone https://github.com/beatrizmoutinho/QaTestProject.git
	powershell -ExecutionPolicy Bypass -File bin/Debug/net8.0/playwright.ps1 install
	
---

## How to Run

	dotnet build
	dotnet test
